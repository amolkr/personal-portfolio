# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, education, and contact information while providing a clean and professional user experience across all devices.

## 🚀 Features

* Modern Dark-Themed UI
* Fully Responsive Design
* Project Showcase Section
* Skills & Technologies Overview
* Education Details
* Contact Form
* GitHub & LinkedIn Integration
* Resume Download Option
* Smooth Navigation Experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git & GitHub


## 🔗 Live Demo

https://amolkumar.netlify.app/

## 📂 Sections Included

### Home

Introduction and professional summary.

### About

Brief overview of my background and interests.

### Skills

Technologies and tools I use for development.

### Projects

Featured projects with descriptions and technologies used.

### Education

Academic background and current studies.

### Contact

Ways to connect with me through email, GitHub, and LinkedIn.

## 🎯 Purpose

This portfolio serves as a central hub for showcasing my work, technical skills, and development journey. It was built to demonstrate my front-end development abilities and provide recruiters and collaborators with an overview of my projects and experience.

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/amolkr/portfolio-website.git
```

2. Navigate to the project folder

```bash
cd portfolio-website
```

3. Open `index.html` in your browser
